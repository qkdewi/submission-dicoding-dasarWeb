My Story

Perkenalkan nama saya Qomariah Kumala Dewi, biasa di panggil ria. Saya berasal dari pulau sumatra. Lebih tepatnya saya dari kabupaten Bungo Provinsi Jambi. Saya lahir tanggal 10 Juni 1999, sehingga umur saya sekarang 20th.

Saat ini saya sedang kuliah di salah satu perguruan tinggi swasta yang ada di kota Semarang. Saya dari Fakultas Teknologi Informasi. Program Studi Teknik Informatika. Awal masuk jurusan ini saya tidak mengerti coding sama sekali. Akan tetapi berkat komunitas di Semarang seperti SANDEC, Programmer Semarang, dll saya termotivasi untuk belajar. Di dukung oleh dicoding, Udemy, Udacity, Petani Kode dan Codepolitan saya semakin bersemangat belajar.
